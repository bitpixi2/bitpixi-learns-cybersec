# BITPIXI LEARNS CYBERSEC

A private Australian cyber-security and responsible AI leadership fieldbook.

The site pairs two authentic CyberBrokers—Woodsy Dusty (#2821) for the technical track and Guarded of Vibrant (#8377) for the human layer—with an 18-mission study checklist, Australian research watchlist, white-hat engagement gate, and future merch concepts.

The original token SVGs, PNG renders, and metadata snapshots live in `public/nft/`.

## Commands

- `npm run dev` starts local development.
- `npm run build` creates the production build.
- `npm test` builds and verifies the rendered fieldbook.
- `npm run lint` runs the code-quality checks.
